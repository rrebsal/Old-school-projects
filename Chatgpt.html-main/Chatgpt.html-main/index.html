Koodikielitaidoni

HTML

HTML (HyperText Markup Language) on merkintäkieli, jota käytetään verkkosivujen rakenteen ja sisällön kuvaamiseen.


Olen opiskellut HTML:ää siitä asti, kun aloitin ammattikoulun, joten olemme ehtineet jo opiskelemaan sitä jo jonkin verran. Olen käyttänyt HTML:ää sivustojen tekemiseen, ja sivusto pelejen tekemiseen.

CSS

CSS (Cascading Style Sheets) on tyylisivukieli, jota käytetään verkkosivujen ulkoasun määrittämiseen.


CSS on semi helppoa, joten uskoisin että osaan suurimman osan CSS:än ominaisuuksia. Olen käyttänyt CSS:ää sivustojen muotoiluun.

JavaScript

JavaScript on ohjelmointikieli, jota käytetään usein verkkosivujen interaktiivisuuden toteuttamiseen.


En ole ehtinyt opiskella JavaScriptiä vielä paljoa, mutta olen oppinut alkeet. JavaScriptillä olen muokannut sivustoja, ja tehnyt komentoja sivustolle.

Python

Python on yleiskäyttöinen ohjelmointikieli, joka on suosittu mm. web-kehityksessä, tietojenkäsittelytieteessä ja tekoälysovelluksissa.

Java

Java on laajalti käytetty, monipuolinen ohjelmointikieli, jota käytetään mm. sovelluskehityksessä ja mobiilikehityksessä.
